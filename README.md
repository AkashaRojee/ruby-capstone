# Catalog of my things
> This is a console app that allows users to manage collections of the things they own. The app allows:

- Add new items.
- Show selected items.
- Preserve user data by saving collections in .json files.

## Built With

- Ruby

## Getting Started

To get a local copy up and running follow these simple example steps.

### Prerequisites

- [Ruby](https://www.ruby-lang.org/en/)

### Setup

- Clone or download this repo on your machine
- Enter project directory

### Run

```rb
ruby main.rb
```
